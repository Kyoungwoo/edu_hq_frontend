도움말

1. 허가작업에 대해 사전 안전조치 등이 미흡하여 공사감독자(감리원)의 보완지시가 있는 경우 미흡사항에 대한 조치 및 공사감독자(감리원)의 확인 후 작업 착수 가능 합니다.
2. 최초 작업허가서는 감독자(감리원)의 승인(승인용 양식참조)을 득해야하며, 승인 후 승인받은 작업과 작업조건(작업내용, 위치 등)이 동일하게 반복되는 경우 둘째날부터는 작업계획 제출(제출용 양식참조)후 작업 가능 합니다.
3. 타워크레인, 건설용리프트, 항타·항발기 작업을 수행할 경우 장비 작업계획서를 필수로 첨부 해 주세요.
4. 첨부 서류는 최초 작업 승인시에만 첨부 합니다. (작업조건이 동일하게 반복되어 작업계획 제출시에는 생략가능) (승인) 안전 작업 허가서(PTW)
