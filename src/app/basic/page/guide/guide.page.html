Library Guide

component

화면에 tag를 직접 넣어 사용하거나, service와 연동하여, 새로운 화면을 띄워주는 역할을 합니다.

Dialog

alert, modal 등 화면에 새로 띄우는 컴포넌트들 입니다. tag를 직접 넣어 사용하지 않고, service를 이용해서 사용합니다.

app-alertservice

alert:present

UI

app-button

앱에 사용되는 버튼 입니다. 기본적으로는 ion-button과 같은 사용법입니다.

default shape:round shape:circle shape:square size:small fill:translucent fill:outline fill:clear
expand:block

정사각형과 원은 사이즈가 달라져도 따라 모양이 틀어지지 않습니다.

shape:circle
style.width
150px shape:square
style.width
150px

아이콘 및 다른 컴포넌트들과 함께 사용할 수 있습니다.

icon:start

icon:top

app-button-footer

이것은 하단에 고정되는 footer 버튼을 쉽게 만들기위한 component 입니다. 페이지 하단을 참고해주세요.

app-card

앱 카드에 주로 사용되는 레이아웃 입니다. 내부 header, title, subtitle, content는 ionic 원본을 사용합니다.

ion-card-header ion-card-title ion-card-subtitle ion-card-content ion-note ion-card-header button:false

app-copy-right

페이지 하단의 로고 및 카피라이트를 표시하기 위한 component 입니다.

app-img

기존 img component 입니다. 다크모드 호환, 이미지 사이즈 조정 등을 하기위해 만들었습니다.
서버에서 받아오는 이미지는 dark mode가 없으므로, 일반 img tag를 사용해야 합니다.

app-logo

페이지 전체에 공유하는 '앱의 로고' 레이아웃 입니다.

app-profile

프로필 이미지를 표시하기 위한 component 입니다. app-input-file 을 상속하고 있어서, 이미지 추가, 삭제를 할 수 있습니다.

app-row / app-col

flex 레이아웃을 짜기 위한 component 입니다.

size: auto size: 200px size: 2 not sized

app-table

테이블 component 입니다. 일반 테이블처럼 thead, tbody, tr, th, td를 사용합니다. 때문에 class를 통해서 속성을 조정하게 됩니다.

# tag class example 1 normal normal normal 2 th,td small use <small> tag 3 tr class.button - 4 th,td class.button - 5 tr,th,td class.text-align-left text align left 5 tr,th,td class.text-align-right text align right
